# Overview

## Ringkasan
GeoMap CRM adalah aplikasi CRM enterprise berbasis native PHP 7.x dengan kemampuan geospasial menggunakan Leaflet + OpenStreetMap.

Database engine yang didukung:
- MySQL
- SQL Server 2019+ (`sqlsrv`)

## Tech Stack
- Backend: PHP 7.x native (tanpa framework)
- Frontend: HTML5, Bootstrap 5, JavaScript (Vanilla + jQuery)
- Peta: Leaflet.js + OpenStreetMap tiles
- API: REST JSON (`/api`)
- Database: MySQL atau SQL Server
